# Safe Browsing Buddy 🛡️

A Chrome Extension that analyzes website URLs to detect phishing and suspicious patterns.

## Features
- ✅ 12-point URL analysis
- ✅ Real-time risk scoring (0-100)
- ✅ Dark cybersecurity UI
- ✅ 100% local processing
- ✅ No data collection
- ✅ Manifest V3 compliant

## Installation
1. Download/clone this repo
2. Open `chrome://extensions/`
3. Enable Developer Mode
4. Click Load Unpacked
5. Select Safe-Browsing-Buddy folder

## Usage
Click extension icon → Automatic analysis → Review risk score

## Analysis Points
1. IP Address Detection
2. Long URLs
3. Excessive Subdomains
4. Suspicious Keywords
5. Domain Impersonation
6. Multiple Hyphens
7. URL Shorteners
8. Missing HTTPS
9. Number Ratio
10. Non-Standard Ports
11. Excessive Parameters
12. Uncommon TLDs

## Risk Levels
- 0-29: Low Risk ✅
- 30-59: Suspicious ⚠️
- 60-100: High Risk 🚨

## Technologies
- Manifest V3
- Vanilla JavaScript
- HTML5 + CSS3

## Privacy
✅ All analysis local
